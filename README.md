Project Description

GoldenHour:
GoldenHour is a web-based platform designed to connect people facing medical emergencies with critical resources in real-time. In Bangladesh, during medical emergencies, people often struggle to find blood donors, locate rare medicines, check hospital bed availability, or get immediate assistance. GoldenHour solves these problems by creating a network that connects patients, donors, hospitals, first responders, and medical resources on a single platform.
